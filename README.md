# Course Critic

## Purpose
As a potential student, I want to research on classes in my field, so that I can see the feedback from former students.

## Description
* As soon as the user enters the site they will see the homepage which shows the categories on the left-side. 
* On the top it demonstrates the links for home, course, about, login and logout. On the right side the website shows the pictures which pertains to the specific category it falls under. 
* Right below the picture is the course name. 
* The picture as well as the name once it is clicked it will take the user to the specific information pertaining that course. Each course will have course name, school, and description. Each course also has a comment section. 
* The user is also able to login through their twitter and github account.
